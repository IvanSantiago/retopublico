Contenido de repositorio
===========

Dentro de este repositorio se encuentran los códigos fuente de los proyectos en eclipse para ejecutar el prototipo de la propuesta de MI MAPPIR, por medio de una importación de GIT, así mismo se encuentra un video demostrativo de la funcionalidad de la propuesta y un archivo que contiene el URL de Dropbox donde se puede realizar la descarga de una máquina virtual realizada para ser ejecutada en Virtual Box.
