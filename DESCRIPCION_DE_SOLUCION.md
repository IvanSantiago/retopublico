Como se soluciona el reto
===========

En esta pequeña demo se muestra un poco la funcionalidad que se desea agregar al sistema MAPPIR esta propuesta contempla 2 partes fundamentales:
1)	Permitir al usuario comentar sobre diferentes situaciones sobre la carretera así como reportar la misma, ya sea de manera automática vía móvil o de manera remota por medio de la versión Web, en esta implantación se planea agregar funcionalidades NATIVAS dentro de las aplicaciones móviles para el uso de ubicación de usuario automáticamente así como mejorar el PERFORMANCE de la aplicación.
2)	Permitir que el usuario del sistema MAPPIR pueda alimentar una base de datos automáticamente si es que así lo desea en donde pueda enviar a sus familiares y conocidos su ubicación o simplemente llevar un seguimiento de sus actividades dentro de la carretera para futura explotación de la información, como es el caso de los transportistas.
El objetivo de este demo es poder mostrar en breve la funcionalidad de las adiciones al aplicativo así como el ver las diferentes partes que se involucran dentro del proceso.
Así como permitir que se muestre de una manera más visual las funcionalidades y el potencial que se puede tener de información para comenzar una iniciativa de “Social Big Data” enfocada en el transporte.
